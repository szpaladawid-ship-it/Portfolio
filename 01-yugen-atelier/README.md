# Yūgen Atelier

A fictional architecture and spatial design studio created as the first complete portfolio case study.

## Project purpose

Yūgen Atelier is designed to demonstrate:

- premium editorial web design,
- multilingual EN/JP localization,
- Webflow CMS structure,
- responsive layouts,
- restrained motion design,
- portfolio storytelling,
- project detail templates.

## Brand profile

- **Name:** Yūgen Atelier
- **Industry:** Architecture, interiors, spatial design
- **Locations:** Tokyo / Oslo
- **Working languages:** English / Japanese
- **Character:** quiet, precise, contemporary, tactile
- **Visual direction:** Japanese restraint combined with Nordic minimalism
