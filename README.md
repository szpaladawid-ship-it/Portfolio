# ShinGiTai Digital Portfolio

Public portfolio and agency-partnership workspace for Dawid Szpala and the emerging ShinGiTai Digital service line.

## Purpose

This repository will contain:

- selected portfolio case studies,
- public demo websites and frontend projects,
- technical capability summaries,
- a white-label agency partner kit,
- delivery standards and workflow documentation,
- reusable project templates.

## Current status

Foundation phase. The repository structure and first portfolio materials are being prepared.

- React
- Next.js
- TypeScript
- Tailwind CSS

- Responsive web design

ShinGiTai Digital is being prepared to support agencies and design studios with reliable white-label frontend implementation, including:

- responsive implementation from approved designs,
- React and Next.js development,
- TypeScript frontend work,
- CMS integration,
- performance optimization,
- maintenance and bug fixing.
